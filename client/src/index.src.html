Infinite

about

Infinite is a relaxing puzzle game where you can continuously expand the galaxy by matching the edges of hexagonal tiles. No need to stress about losing — there are always valid placements.

Infinite

how to play

! Edge users: The game will not work properly if visual search is enabled.

Rotate tile.

Use buttons around preview of next tile, or triple-tap to rotate clockwise and double-tap to rotate counter-clockwise.

Place tile.

Tap to place tile. Edges need to match with existing tiles.
There will always be at least one matching location.

Adjust view.

Pan by dragging.

Zoom by using + and - buttons.
